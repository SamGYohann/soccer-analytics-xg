# Premier League Striker Efficiency Analysis (2024–2025)

Analysis of Premier League striker performance using Python. Compares expected goals (xG/90) to actual goals scored per 90 minutes to identify over- and under-performers.

## Tools & Libraries
- Python (Pandas, Matplotlib, SciPy)
- Linear regression and correlation analysis
- Data visualization (histogram, scatter plot with regression line)

## Key Findings
- Correlation of r = 0.57 between xG/90 and G/90 (p-value: 1.3e-14)
- Identified strikers who consistently over/under-perform relative to chance quality
- Findings framed for two stakeholders: coaching staff and recruitment/scouting

## Data Source
Kaggle / FBref — Premier League player stats, 2024–2025 season
